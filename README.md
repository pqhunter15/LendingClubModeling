# Lending Club Modeling
## Purpose
This notebook was created as part of a Final Project for a model risk management course. The objective was to model lending club data to predict probability of default, and analyze the results with a focus on model risk management. 
As models employed by financial institutions to make decisions which impact clients fall under strcit scrutiny by auditors, The post modeling analysis focuses on model explainability including:
  - Feature importance
  - Partial Dependence Plots
  - SHAP global and local explainability
  - Probability Calibration.
